The website contact forms use FormSubmit: https://formsubmit.co/cohare@tateandfoss.com
This submits through HTTPS and emails Chantal. On the first submission, FormSubmit may send Chantal an activation email that must be confirmed.
For a more controlled long-term setup, replace the form action with a Netlify Forms endpoint, Formspree endpoint, or a small serverless function. Do not put SMTP passwords or API keys in front-end HTML.
